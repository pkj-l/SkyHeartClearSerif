# 空心晴宋体 / 空心晴明體 / Sky Heart Clear Serif

基于[《霞鹜铭心宋》](https://github.com/lxgw/LxgwHeartSerif)[《霞鹜新致宋＋》](https://github.com/lxgw/LxgwNeoZhiSong)[《致一宋体》](https://github.com/RiyipuSipino/ZhiyiSerif)及其原始字体的中文宋体。

将《霞鹜新致宋＋》和《致一宋体》字面横向压缩 85%，并增加圆角，更加圆润、轻盈。

本字体包含简繁两版，简体版“空心晴宋体（Sky Heart Clear Serif SC）”优先采取《霞鹜新致宋＋》字形，繁体版“空心晴明體（Sky Heart Clear Serif TC）”优先采取《致一宋体》字形。

> [!NOTE]
> 本字体**不会**与上游字体同步更新。
